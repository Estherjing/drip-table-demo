##  development

1. Clone

   ```sh
   git clone https://github.com/JDFED/drip-table-demo.git
   ```

2. Install dependencies

   ```sh
   lerna bootstrap
   ```

3. build independecies

   > yarn

   ```sh
   yarn run build
   ```

   > npm

   ```sh
   npm run build
   ```

4. Run project

   ```sh
   yarn start
   ```

* visit http://localhost:3000
